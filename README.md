# Invisible Captcha Example

This repository demonstrates how to implement an **invisible CAPTCHA** using the [ALTCHA](https://altcha.org) widget with React.

It builds on the official starter template: [altcha-org/altcha-starter-react-ts](https://github.com/altcha-org/altcha-starter-react-ts)

This setup ensures users aren't interrupted unless absolutely necessary - improving UX without sacrificing protection.

Read the full documentation: [Invisible CAPTCHA Documentation](https://altcha.org/docs/v2/invisible-captcha)
